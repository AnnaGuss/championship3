#  Championship3

![Vite](https://img.shields.io/badge/Vite-8.x-purple)
![Node](https://img.shields.io/badge/Node-%3E%3D22-green)
![License](https://img.shields.io/badge/license-MIT-blue)

Современный frontend-проект, собранный с использованием Vite.
Проект демонстрирует базовую настройку сборки, линтинга и работы со стилями (SCSS).

##  Установка и запуск

### Установка зависимостей

```bash
npm ci
```

### Запуск в режиме разработки

```bash
npm start
```

После запуска открой адрес из терминала (обычно: http://localhost:5173)

---

##  Сборка проекта

```bash
npm run build
```

Результат сборки будет в папке:

```
dist/
```

---

##  Линтинг

```bash
npm run lint
```

Проверяет:

* JavaScript (ESLint)
* SCSS (Stylelint)

##  Технологии

*  Vite
